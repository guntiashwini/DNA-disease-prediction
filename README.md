# 🧬 DNA Disease Prediction using Hybrid AI (ML + DL)

## 📌 Overview

This project focuses on predicting genetic diseases using DNA sequence data by combining Machine Learning (ML) and Deep Learning (DL) techniques.
The system analyzes DNA patterns and provides disease prediction along with risk levels.

---

## 🎯 Features

* Predicts diseases from DNA sequences
* Uses Hybrid AI (ML + DL models)
* Provides risk score and category
* Visualizes DNA composition (GC vs AT content)

---

## ⚙️ Technologies Used

* Python
* NumPy, Pandas
* Scikit-learn
* TensorFlow / Keras
* Matplotlib

---

## 🧪 Methodology

1. Data Collection (NCBI, GenBank datasets)
2. Data Preprocessing (Cleaning and encoding DNA data)
3. Feature Extraction (k-mer, GC content)
4. Model Training (ML and DL models)
5. Prediction and Evaluation

---

## 📊 Results

* Predicts diseases like Hepatitis
* Generates risk score and classification
* Provides visualization graphs
